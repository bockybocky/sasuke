# Sasuke 佐助寫輪眼

A data-readiness steward for agent systems: every raw file in your knowledge vault is agent-readable **before** it is needed — registry-driven weekly sweep, deny-by-default global census, contained sidecars, index admission gates, and loud failure.

Design hardened by dual cross-vendor red teams (the naive census design FAILED review; the ten fixes are baked into SKILL.md).

Companion repos: [`shanya-shuyuan`](https://github.com/bockybocky/shanya-shuyuan) (book course engine this was born to feed).

Methodology-first repo: SKILL.md carries the full design. Engine code is environment-specific (Windows schtasks + local vault); port the design, not the paths.
